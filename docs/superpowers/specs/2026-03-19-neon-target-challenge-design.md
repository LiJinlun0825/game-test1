# Neon Target Challenge - 游戏设计文档

**版本**: 1.0.0 | **日期**: 2026-03-19 | **状态**: 已批准

## 1. 游戏概述

**Neon Target Challenge** 是一款跨平台的霓虹赛博风格打靶射击游戏，采用纯 HTML/CSS/原生 JavaScript 开发，支持桌面和移动设备。

### 1.1 核心体验
- 精确点击反馈（靶心判定 + 连击奖励）
- 渐进式难度曲线（5 关 +，速度/大小/复杂度递增）
- 霓虹赛博视觉风格（发光效果、粒子特效）
- 完整音效系统（Web Audio API）
- 本地排行榜（Top 10）

## 4. 靶子系统

### 4.1 靶子类型

| 类型 | 颜色 | 速度 | 大小 | 生命值 | 得分 | 特殊效果 |
|------|------|------|------|--------|------|----------|
| 标准靶 | 青色 #00FFFF | 基准 | 60px | 1 | 10 | 无 |
| 快速靶 | 黄色 #FFFF00 | 2x | 40px | 1 | 25 | 闪现出现 |
| 重甲靶 | 橙色 #FF8800 | 0.5x | 80px | 3 | 50 | 需多次点击 |
| 奖励靶 | 粉色 #FF00FF | 1.5x | 30px | 1 | 100 | 时间 +5s |
| 惩罚靶 | 红色 #FF0000 | 1.5x | 50px | 1 | -50 | 连击清零 |

### 4.2 靶子行为模式

```javascript
// 移动模式
const MovePatterns = {
  LINEAR: 'linear',      // 匀速直线
  BOUNCE: 'bounce',      // 反弹
  CURVE: 'curve',        // 正弦曲线
  RANDOM: 'random',      // 随机变向
  HOMING: 'homing',      // 跟随鼠标（高阶）
};

// 出现模式
const SpawnModes = {
  NORMAL: 'normal',      // 从边缘滑入
  POP: 'pop',            // 随机位置弹出
  FADE: 'fade',          // 渐显
};
```

### 4.3 环数判定

```
┌─────────────────────────────────────┐
│              外环 (3 分)             │
│  ┌─────────────────────────────┐    │
│  │           内环 (5 分)        │    │
│  │  ┌─────────────────────┐    │    │
│  │  │     靶心 (10 分)      │    │    │
│  │  │        ★            │    │    │
│  │  └─────────────────────┘    │    │
│  └─────────────────────────────┘    │
└─────────────────────────────────────┘
```

判定逻辑：
- 计算点击位置与靶心的距离
- 距离 < 半径 × 0.3 → 靶心 (10 分)
- 距离 < 半径 × 0.6 → 内环 (5 分)
- 距离 < 半径 × 1.0 → 外环 (3 分)

### 6.3 粒子特效

击中特效参数：
- 粒子数：8-12 个/击中
- 寿命：300-500ms
- 速度：随机 2-5 单位
- 颜色：基于靶子类型
- 重力：轻微向下

## 7. 音效设计

### 7.1 音效列表

| 音效 | 触发时机 | 描述 |
|------|----------|------|
| BGM | 游戏进行 | 赛博朋克风格循环背景音乐 |
| Hit | 击中靶子 | 清脆电子音（音调随环数变化） |
| Headshot | 靶心击中 | 高音调特殊音效 |
| Spawn | 靶子出现 | 短促 pop 音 |
| Combo | 连击达成 | 上升音阶（5/10/15 连击） |
| TimeUp | 时间到 | 低沉警告音 |
| LevelUp | 关卡完成 | 胜利旋律 |
| GameOver | 游戏结束 | 下降音阶 |

### 7.2 音频实现

使用 Web Audio API：
- 预加载所有音效
- 支持音量调节
- 动态混音（BGM 音量自动降低当音效播放时）
